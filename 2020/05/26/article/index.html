    学习了廖雪峰老师的git教程

    2020-05-26
    step1 - 安装
    • 三个库
      1是工作区 working directory
      2是暂存区 stage
      3是版本库 repository
    • 对应三个动作

    1.从工作区提交至暂存区使用add
    2.从暂存区提交至版本库使用commit

    安装完成后记得进行配置
    $ git config –global user.name “Your Name”
    $ git config –global user.email “email@example.com“

    step2 - 初使化

    使用以下命令将创建的文件目录变成repo
    $ git init

    • 初始化一个Git仓库，使用git init命令。
    • 添加文件到Git仓库，分两步：

    使用命令git add ，注意，可反复多次使用，添加多个文件；
    使用命令git commit -m ，完成。

    • 要随时掌握工作区的状态，使用git status命令。

    • 如果git status告诉你有文件被修改过，用git diff可以查看修改内容。（只能看工作区文件的修改情况）

      git diff HEAD –用于查看工作区与版本库的文件差别

    step3 - 版本回退
    • HEAD指向的版本就是当前版本，因此，Git允许我们在版本的历史之间穿梭，使用命令git reset –hard commit_id。
    • 穿梭前，用git log可以查看提交历史，以便确定要回退到哪个版本。

      git log –pretty=oneline

    • 要重返未来，用git reflog查看命令历史，以便确定要回到未来的哪个版本。
    • git status 可以查看文件的修改状态

      每一步都可以使用 git restore进行回退
